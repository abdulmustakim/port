# Abdul Portfolio Website

A modern, responsive personal portfolio website for Abdul Mustakim Molla.

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/abdul-cv.txt`

## Features

- Responsive design
- Sticky navbar
- Mobile hamburger menu
- Dark/light mode
- Typing animation
- Scroll progress bar
- Animated skill bars
- Project filter system
- Services section
- Education timeline
- Resume download button
- Contact form demo
- Floating WhatsApp button
- Back-to-top button
- SEO meta tags

## How to Run

1. Extract the ZIP file.
2. Open `index.html` in Chrome.
3. Edit the files in VS Code.

## Important Customization

Update these before showing to clients:

- WhatsApp number in `index.html`
- LinkedIn and Instagram links
- Project live demo links
- Project source code links
- Your real phone number
- Your real CV file
